描述

2053年夏，讲究的包大爷决定给孩子们演示自己最喜欢的手冲咖啡。他一边研磨着豆子一边说，“这种手法，你们绝遇不见。这是以前Banker的基本功，不像你们现在喝那些成品的一罐一罐，粗制滥造，那种咖啡还要的吗？”


手摇的磨豆机还在“克拉克拉”作响，像一种远古的戏法。

老包闻着味道说“要选这种烘焙度偏深的单品豆子，它里面有二氧化碳，到时候悬浮在表面的咖啡粉会有一个保温的作用，就跟闷煮一样。”


孙子把下巴搁在台子上，看着老包把折好的滤纸放在冰瞳滤杯中。他拿起手冲壶里的热水顺着滤杯圆形倒入，见注水处的咖啡粉股的像火山喷发一样。


“这叫火山冲，对好的单品豆要求很高。这个是也门产的Mokha Mattari，高海拔才能有的咖啡。酒香、水果干味，深度烘焙有巧克力的苦甜。水流得稳得细，高温萃取才能得到醇厚结实的body，还能有强烈回甘”说着把滤杯给孙子拿近了瞧，“这个样子，等会你就能喝出像法兰丝绒般醇厚结实的感觉”。


“这个要分三次冲煮，分三段萃取”。泡好后，老包拿在手中摇一摇，递给孙子尝。


“嗯，爷爷，苦但是好喝，入口柔顺，有肉桂和巧克力的香气，酸度不强，有杏仁香”砸吧砸吧嘴巴，“喝下去，嘴巴里还有乌梅和奶香。好厉害！”


老包笑眯起了眼，想到碳中和已经提前7年实现，这种不经低碳工业流程的咖啡豆是不是已经成为了一种艺术，应该束之于高阁，流传成传说。某种高雅，已经随着时代被人间淡忘，尽管人间琐事与40年前并无二致，自己所经历的，是不是自己的孙儿也还会经历。


叹气着，老包冲了第二泡。

突然身后响起了

“故事的小黄花，从出生那年就飘着，

童年的荡秋千，随记忆一直晃到现在

Re So So Si Do Si La So

La SI Si Si Si La Si La So

…”


水冲进咖啡粉，花瓣从天空掉落。

消失的下雨天，教室的那一间。

2053年夏，2003年夏。

吴承恩（约1500年—1582年），字汝忠，号射阳居士，又称射阳山人，南直隶淮安府山阳县河下（今江苏淮安）人（一说安徽桐城），祖籍安东。明代文学家。吴承恩自幼敏慧，博览群书，尤其喜爱神话故事。擅长绘画、书法，多才多艺。在科举中屡遭挫折，嘉靖中举为岁贡生。嘉靖二十八年（1549年）迁居南京，以卖文为生。嘉靖三十九年（1560年）任浙江长兴县丞，不久辞官归去。由于宦途困顿，晚年绝意仕进，闭门著书，终老于家。近现代学者一般认为吴承恩是中国四大古典名著之一的《西游记》的最后完成者。

雨人，《雨人》是巴瑞·莱文森执导的一部剧情电影，由达斯汀·霍夫曼、汤姆·克鲁斯、瓦莱莉·高利诺、迈克尔·D·罗伯茨、邦尼·亨特、贝丝·格兰特主演。影片中，查理发现父亲将遗产留给了患自闭症的哥哥雷蒙，便想骗取这笔财富，并计划利用哥哥超强的记忆力去赌博赚钱。但在此过程中，血缘的亲情打破了原有的疏离，真挚动人的手足之情取代了查理原先只求一己利益的私心。影片于1988年12月12日在美国上映，获第61届奥斯卡最佳影片。
